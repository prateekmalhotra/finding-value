**F) TERMINAL VALUE (REVISED METHODOLOGY)**

**17) Correcting the Terminal Value Logic:**
The original model's primary issue was using an exit multiple on a business with negative cash flow, which is logically unsound. A buyer would not pay a premium for a stream of negative cash flows. The solution is to calculate a **normalized free cash flow** for the terminal year, assuming the company has transitioned from its high-growth/high-investment phase to a mature, steady state.

*   **Step 1: Project Terminal Year (Year 6) Earnings**
    *   EBIT_Yr6 = EBIT_Yr5 * (1 + g) = $242.60M * (1.025) = $248.67M
    *   NOPAT_Yr6 = EBIT_Yr6 * (1 - Tax Rate) = $248.67M * (1 - 0.22) = **$193.96M**

*   **Step 2: Normalize Reinvestment for a Mature State**
    *   In a mature state, a company's capex should primarily cover the depreciation of its assets. A realistic assumption is that **Terminal Capex = Terminal D&A**. This implies the net investment required to generate the 2.5% terminal growth is minimal, funded by efficiency gains.
    *   D&A_Yr6 = D&A_Yr5 * (1.025) = $148.99M * 1.025 = $152.71M
    *   SBC_Yr6 = SBC_Yr5 * (1.025) = $7.05M * 1.025 = $7.23M

*   **Step 3: Calculate Normalized Terminal FCFF**
    *   Normalized FCFF = NOPAT + D&A - Capex - SBC
    *   Normalized FCFF = $193.96M + $152.71M - $152.71M - $7.23M = **$186.73M**

*   **Step 4: Calculate Terminal Value using Gordon Growth Method**
    *   **Terminal Value (TV):** TV = Normalized FCFF / (WACC - g) = $186.73M / (4.88% - 2.5%) = **$7,845.8M**

**18) Exit Multiple Cross-Check:**
